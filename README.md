### Training-Tracker
Your every day workout routine.

Purpose of this app is to let you focus on you fitness training and let the app keep the track.

With this app you do not need to remember the day or schedule of your training as this app is configured in a way that keeps track of the day of a week and the exercises scheduled for the day.

Tech satck used:
1. HTML5
2. CSS
3. Bootstap
4. API
5. Javascript
